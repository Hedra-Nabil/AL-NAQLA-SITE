🚀 Quick Start Guide

Get started with AL-NAQLA in just a few simple steps. Choose your platform below and follow the detailed setup guide.

📱 Android Setup

Complete setup guide for Android devices

  1. 1
    Download Application

    Download the latest AL-NAQLA APK file from our official release page

    File size: ~25MB | Version: 2.1.0
  2. 2
    Enable Installation

    Go to Settings → Security → Enable "Install from Unknown Sources"

    This is safe for AL-NAQLA installation
  3. 3
    Install Application

    Tap the downloaded APK file and follow installation prompts

  4. 4
    Launch & Setup

    Open AL-NAQLA, create your account, and complete the initial setup wizard

  5. 5
    Start Managing

    Begin managing your transportation operations with our comprehensive tools

🍎 iOS Setup

Complete setup guide for iOS devices

  1. 1
    Request Access

    Contact our team to request access to the iOS beta version

    Processing time: 24-48 hours
  2. 2
    Install TestFlight

    Download TestFlight from the App Store if not already installed

  3. 3
    Accept Invitation

    Open the TestFlight invitation link sent to your email

  4. 4
    Install AL-NAQLA

    Install AL-NAQLA through TestFlight

  5. 5
    Setup Account

    Create your account and start managing your transportation business

🖥️ Windows Setup

Complete setup guide for Windows desktop

  1. 1
    Download Installer

    Download the AL-NAQLA Windows installer (.exe file)

    File size: ~45MB | Windows 10+ Required
  2. 2
    Run Installer

    Right-click the installer and select "Run as administrator"

  3. 3
    Follow Setup Wizard

    Complete the installation wizard with default settings

  4. 4
    Launch Application

    Open AL-NAQLA from the desktop shortcut or Start menu

  5. 5
    Configure & Sync

    Set up your workspace, connect to cloud services, and sync your data
